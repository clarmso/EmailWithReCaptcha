"Where Am I?" Game

This is a game that quizzes the players' geographic knowledge around the world. I designed the game, collected the location data and compile the (supposingly funny 😄 but not offensive) memes. This project was a part of the web development course offered by BrainStation.

  • Tech Stack: Node.js, React.js, Google Maps API, CSS/LESS, Gulp, Livereload
  • Source: Github Repo
  • Date: Summer 2015
